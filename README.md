# Codex Showcase

A TanStack Start web app that imitates the Codex native app UI for a future Codex ambassador project showcase.

The first screen intentionally mirrors the Codex empty-thread state: project sidebar, top-right window controls, centered prompt composer, local context strip, and connector cards.

## Development

```bash
npm install
npm run dev
```

The dev script requests port `3000`. If that port is already in use, Vite will choose the next available port.

## Verification

```bash
npm run lint
npm run build
```
